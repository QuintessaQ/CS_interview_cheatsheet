# names of large numbers
- 1 KB = 1024 bytes = 10^3 bytes | kilobytes
- 1 MB = 1024 KB = 10^6 bytes | megabytes
- 1 GB = 10^9 bytes | gigabytes
- 1 TB = 10^12 bytes | terabytes
- 1 PB = 10^15 bytes | petabytes
